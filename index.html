software-escolar-pec3 by software-escolar-comunidad

Software escolar libre

Software escolar pensado para la automatización de procesos de inscripción, matriculación, calificación, asignación, planificación y demas, que realizan la administración, el profesorado y los estudiantes

Dependencies

Necesitan tener apache2. php5 y una base de datos mysql para utilizar el software. Adicionalmente habilitar el modulo php5 en apache2

Install

Descromprimir el fichero en un directorio del DocumentoRoot (generalmente /var/www) crear la bbd con el script crebas.sql (nombre de base de datos "sel") Se esta trabajado con mas información en la Wiki
